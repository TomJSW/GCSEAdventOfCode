GCSE AOC!

Welcome to GCSE Advent Of Code!

An A Level project developed by Tom Swainston

Within this website are a set of mixed difficulty coding challenges that cover all aspects of the "coding" side of your OCR GCSE Computer Science qualification.

Previously there was a coding project that contributed to your Computer Science qualification; however, this has been recently removed from the course completely due to difficulties in moderation. As a result of this, there is no official coding aspect on your course. Coding is an essential skill of Computer Science!

This website will ensure you stay on top of your coding ability as well as ensuring that you are ready for any coding-related questions that may appear on your written exam.

Good luck.
